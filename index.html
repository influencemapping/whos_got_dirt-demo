Who's Got Dirt? Demo

This is a demonstration of the Who's got dirt? API. The Who's got dirt? API provides a single access point to multiple APIs of influence data on the web. It proxies requests to the supported APIs, so that users only need to learn a single request format and a single response format. Read the documentation to learn more about the API and how you can use it in your own projects.

This demo queries only entities (people and organizations) and exposes only 5 filters for entities, not all 25 implemented by the Who's got dirt? API and its supported APIs. In addition to entities, the Who's got dirt?1 API supports relations between entities (for example, a person holding an office in an organization) and lists of entities (for example, a corporate grouping).
